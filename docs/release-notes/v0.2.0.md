# QuietScribe v0.2.0

This release adds local live microphone transcription while keeping the existing file transcription workflow intact.

## Added

- New Live Transcription page with microphone selection, start, pause/resume, stop, clear, copy, and `.txt` export.
- WASAPI microphone capture with 16 kHz mono PCM chunking for the transcription worker.
- Persistent `worker.py live` mode that loads the selected faster-whisper model once and accepts rolling audio chunk commands over JSON Lines.
- Live JSONL events for `ready`, `live_partial`, `live_segment`, `live_progress`, `live_error`, and `live_stopped`.
- Partial current-line text plus stable timestamped transcript segments.
- Overlap deduplication to reduce repeated words from rolling windows.
- Tiny and base faster-whisper model options for lower-latency live transcription.
- Live transcription docs with latency, model, hardware, and protocol notes.

## Changed

- The desktop app now requests microphone capability for packaged builds.
- The model downloader User-Agent and app package version are updated to `0.2.0`.

## Notes

- Live transcription stays local; no cloud transcription has been added.
- Accuracy and latency depend on model size and hardware. Tiny, base, and small models are recommended for live mode.
- Real-time speaker diarization is not included. Existing speaker detection remains a file/post-processing workflow.
