# QuietScribe v0.1.6

This release keeps the working portable packaging and fixes the missing taskbar icon path.

## Fixed

- Applies the QuietScribe icon from an absolute file path at startup.
- Sets both small and large Win32 window icons so the running app can show the right taskbar/window icon.

## Changed

- Public releases are portable-only for now because Windows Smart App Control can block unsigned setup installers.
- The Inno Setup script is still in the repo for local/manual installer builds, but setup artifacts are not attached to this release.
