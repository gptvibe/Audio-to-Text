# QuietScribe v0.2.1

This release reorganizes transcript history into a persistent local sidebar so recent transcripts are available without opening a separate History page.

## Added

- Persistent left transcript sidebar with New Transcription, local history search, and recent transcript list.
- Click-to-load history entries directly into the main transcript editor.
- Sidebar refresh after creating or deleting a transcript.
- Compact sidebar behavior for narrow windows.

## Changed

- Removed History from the main navigation tabs.
- Moved Live Transcription, Models, and Settings into the shell sidebar.
- Updated README and release metadata to `0.2.1`.

## Notes

- Transcript history remains local-only.
- No cloud sync or AI chat features have been added.
