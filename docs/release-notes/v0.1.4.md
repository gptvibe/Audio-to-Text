# QuietScribe v0.1.4

This release fixes a portable packaging regression from v0.1.3.

## Fixed

- Removed the custom native launcher introduced in v0.1.3 because Smart App Control could block it on some Windows 11 systems.
- Reverted the portable zip and installer to the standard WinUI/.NET publish executable layout that worked in v0.1.0 through v0.1.2.
- Kept the polished QuietScribe app icon and bundled transcription worker runtime.

## Notes

- The portable zip now shows the normal support files beside `QuietScribe.exe` again. This is intentional for v0.1.4 so Windows sees the app through the proven executable path.
- If v0.1.3 was blocked on your machine, use this v0.1.4 build instead.
