# QuietScribe v0.1.1

This release fixes the first-run transcription failure from v0.1.0.

## Fixed

- Bundles an embedded Python 3.12 runtime in the setup and portable builds.
- Bundles the core `faster-whisper` transcription dependency stack.
- The app now prefers the bundled worker runtime automatically before falling back to system Python.
- Updated setup/troubleshooting docs so basic transcription no longer asks users to install Python packages by hand.

## Notes

- Speaker diarization remains optional and is not bundled by default because pyannote/Torch is much larger and often requires gated Hugging Face access.
- Download `QuietScribe-v0.1.1-win-x64-setup.exe` for the installer or `QuietScribe-v0.1.1-win-x64-portable.zip` for the portable build.
