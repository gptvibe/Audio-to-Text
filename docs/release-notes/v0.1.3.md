# QuietScribe v0.1.3

This release improves the first impression of the app package.

## Added

- New QuietScribe app icon: a luminous Q-shaped speech bubble with waveform detail.
- New native portable launcher at the root of the portable zip.

## Changed

- Portable zip layout is now cleaner:
  - `QuietScribe.exe` at the root.
  - Runtime files, DLLs, JSON files, worker runtime, and dependencies inside the `App` folder.
- Installer now uses the same clean launcher layout internally.

## Notes

- The root `QuietScribe.exe` is a small native launcher that starts `App\QuietScribe.exe`.
