# QuietScribe v0.1.0

Initial preview release of QuietScribe, a local-first Windows speech-to-text app.

## Highlights

- Modern WinUI 3 desktop shell.
- Local Whisper-compatible transcription through a Python `faster-whisper` worker.
- Hugging Face model downloads with resume support and secure token storage.
- CPU/GPU/NPU hardware detection with safe CPU fallback.
- Drag-and-drop file transcription for common audio/video formats.
- Optional speaker labels when diarization dependencies and gated access are available.
- Transcript editing, search, copy, local history, and `.txt` export.

## Portable Build

Download `QuietScribe-v0.1.0-win-x64-portable.zip`, extract it, and run `QuietScribe.exe`.

Python worker dependencies are not bundled yet. For transcription, install Python 3.10+ and the worker requirements from `workers/transcription-worker/requirements.txt`.
